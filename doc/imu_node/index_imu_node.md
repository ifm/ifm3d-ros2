# IMU node

The `ifm3d-ros2` package provides an Inertial Measurement Unit (IMU) node, which exposes the data captured by the IMU built into the VPU.

For device-level IMU background information and frame/reference-point details, see the [Transforms](imu_transforms.md) section.
